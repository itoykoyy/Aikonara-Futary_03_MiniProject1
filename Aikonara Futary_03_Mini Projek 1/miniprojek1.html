A single-page aplication (SPA)s is a web application or website that interacts with the user by dynamucally rewriting the current page rather than loading entire new pages from a server. This apporoach avoids interruption of the user experience between successive pagez, making of the user experience between successive pages, making the application behave more like a desktop aplication
